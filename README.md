### 環境設定
`.env` ファイルに下記の環境変数を設定
- USER_NAME ( e.g "r-sekishita" )
- USER_HUE ( e.g 300 )
- URL ( e.g "https://kani-life-m6xtjae7da-an.a.run.app/api/command" )

### 始め方・終わり方
- `cargo run` で 🦀 が スポーンする（生まれる）
- `q` で終了

### 操作方法
- `l`: Turn left
- `r`: Turn right
- `←`: Walk left
- `→`: Walk right

### 注意点
- キーの長押しはしないように😡
